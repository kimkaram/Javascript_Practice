자바스크립트 작성 위치

  1. html이 제공하는 시작태그에 on이벤트 속성을 추가해서, 바로 inline 방식으로 자바스크립트 소스를 작성하고 실행되게 할 수 있다.
  2. script 태그 안에 함수를 작성해서 태그와 on이벤트 속성으로 연결 구동되게 할 수 있다.
  3. script 태그는 html 문서 안에서 여러 번 사용할 수 있고, 작성 위치도 제한이 없다.
